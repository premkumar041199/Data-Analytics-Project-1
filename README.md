
# Data Analytics Project: End-to-End Analysis & Dashboard

# Overview

This project demonstrates a complete data analytics workflow, starting from raw data loading to insights presentation.
The goal is to analyze the dataset, extract meaningful insights using Python and SQL, and present results through an interactive Power BI dashboard, a written report, and a presentation.

The project highlights skills in data cleaning, exploratory data analysis (EDA), SQL querying, data visualization, and business-focused storytelling.

# Tools & Technologies

- Python (Pandas)
- MySQL (data storage and SQL queries)
- Power BI (interactive dashboard)
- Gamma (presentation creation)
- GitHub

# Project Steps
1. Data Loading
 - Imported the dataset into Python using Pandas
 - Performed initial data inspection
2. Exploratory Data Analysis (EDA)
 - Analyzed data distributions and trends
 - Identified missing values and outliers
 - Created visualizations to understand patterns
3. Data Cleaning
 - Handled missing and inconsistent values
 - Removed duplicates
 - Standardized data formats
4. SQL Analysis
 - Loaded cleaned data into a MySQL server
 - Wrote SQL queries to answer business questions
 - Extracted aggregated insights for reporting
5. Dashboard Development
 - Built an interactive Power BI dashboard
 - Included key KPIs, filters, and visual insights
6. Reporting & Presentation
 - Created a structured analytical report
 - Designed a presentation using Gamma to summarize findings and insights
